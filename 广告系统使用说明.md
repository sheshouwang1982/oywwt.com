# 广告管理系统使用说明

## 📋 系统概述

已为您的网站创建了完整的广告管理系统，包括：
- **文章页面**：7个广告位（顶部1个 + 中部5个 + 底部1个）
- **首页**：3个广告位（顶部1个 + 中部1个 + 底部1个）

## 🚀 快速开始

### 1. 打开广告管理页面

在浏览器中打开：`d:\web1\ad-manager.html`

或直接访问：`http://你的域名/ad-manager.html`

### 2. 管理广告

1. **切换标签页**：
   - 点击 "文章页面广告" 管理文章广告
   - 点击 "首页广告" 管理首页广告

2. **添加广告代码**：
   - 在相应广告位的文本框中粘贴广告代码
   - 支持任何HTML广告代码（Google AdSense、百度联盟等）
   - 点击 "保存" 按钮保存单个广告
   - 或点击右下角 "💾 保存所有广告" 一次性保存

3. **预览广告**：
   - 点击 "预览" 按钮查看广告效果

4. **清空广告**：
   - 点击 "清空" 按钮删除广告代码

### 3. 更新网站

每次保存广告后，系统会自动下载新的 `ads.js` 文件：
1. 将下载的 `ads.js` 文件覆盖到网站根目录 `d:\web1\ads.js`
2. 刷新网站页面即可看到新广告

## 📍 广告位说明

### 文章页面广告位（7个）

| 广告位ID | 位置 | 说明 |
|---------|------|------|
| article_top | 顶部 | 文章标题下方 |
| article_mid1 | 中部1 | 文章内容前20%处 |
| article_mid2 | 中部2 | 文章内容前40%处 |
| article_mid3 | 中部3 | 文章内容前60%处 |
| article_mid4 | 中部4 | 文章内容前80%处 |
| article_mid5 | 中部5 | 文章内容末尾前 |
| article_bottom | 底部 | 文章内容结束后 |

### 首页广告位（3个）

| 广告位ID | 位置 | 说明 |
|---------|------|------|
| home_top | 顶部 | Featured stories之前 |
| home_middle | 中部 | Featured stories 和 Trending now 之间 |
| home_bottom | 底部 | 页面底部，Footer之前 |

## 💡 使用技巧

1. **广告代码格式**：
   - 支持 `<script>` 标签
   - 支持 `<ins>` 标签（AdSense）
   - 支持任何HTML代码

2. **最佳实践**：
   - 不要在所有广告位都放广告，会影响用户体验
   - 建议使用2-4个广告位即可
   - 顶部和底部广告效果最好

3. **测试建议**：
   - 先在一个广告位测试
   - 确认显示正常后再添加其他广告
   - 使用预览功能检查广告样式

## 📁 文件说明

- `ad-manager.html` - 广告管理页面
- `ad-manager.js` - 管理系统脚本
- `ads.js` - 广告配置文件（自动生成）

## ⚠️ 注意事项

1. **不要手动编辑 `ads.js`**，该文件由系统自动生成
2. **每次修改广告后**，记得下载新的 `ads.js` 并上传到服务器
3. **广告数据保存在浏览器**，清除浏览器数据会丢失配置
4. **建议定期备份**广告配置
